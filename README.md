# shop_exam

Учебный шаблон: PHP + MySQL, роли, CRUD, регистрация.

## Скачать одной командой

```bash
git clone <URL-ТВОЕГО-РЕПОЗИТОРИЯ>.git
```

Пример:

```bash
git clone https://github.com/USERNAME/shop_exam.git
```

## Установка БД

Импортируй `full_db.sql` в MySQL (Workbench или `mysql < full_db.sql`).

Настрой подключение в `var2/api/config.php` (хост, имя БД `shop_exam`, логин, пароль).

Размести содержимое `var2/` в корне веб-сервера (например `/var/www/html/`) или настрой виртуальный хост.

## Вход (из дампа)

- Администратор: `ida` / `2045`
- Менеджер: `leh` / `1234`
- Исполнитель: `vlt` / `2707`

После регистрации новый пользователь получает роль исполнителя.
